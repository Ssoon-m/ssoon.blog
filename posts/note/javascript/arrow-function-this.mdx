> 화살표 함수에서 this가 바인딩 되는 동작을 이해하기 위해선 먼저 스코프 체인 동작에 대해 먼저 알아야 한다.

## Js 스코프 체인
JavaScript에서는 어떤 식별자를 찾을 때 현재 스코프에서 그 식별자가 없으면 바로 상위 스코프 검색한다. (스코프 체인)
점점 상위 스코프로 타고 올라가다가 해당 식별자를 찾으면 탐색을 멈춘다.

## 화살표 함수의 this 바인딩
화살표 함수에서의 this 바인딩 방식도 이와 유사하다.
화살표 함수에서의 this는 존재하지 않으므로 this를 찾기 위해 상위 스코프를 탐색하게 된다.

그런데 화살표 함수가 선언될 시점에서 상위 스코프의 this가 바인딩이 된다고 생각하는 사람들이 있다.
실행컨텍스트가 생성 될 때 화살표 함수의 상위 스코프의 this가 어떻게 결정되는지에 따라서 화살표 함수의 this가 결정이 된다.
